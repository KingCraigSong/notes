# 分割清理脚本 auto_clean.sh

```bash
#!/bin/bash 
DIR=日志地址 
d=`date +%Y%m%d` 
cd $DIR cp catalina.out catalina.out.bak${d} 
echo "" > catalina.out 
find ./ -mtime +7 -name "catalina.out.bak*" -exec rm -rf {} \;
```

# 定时任务

 进入编辑
```bash
crontab -e
```

添加任务
这里的设置是每天23点59分执行 auto_clean.sh 文件进行数据清理任务了。
```bash
59 23 * * * /opt/soft/log/auto_clean.sh >/dev/null 2>&1
```
