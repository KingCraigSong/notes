# 进阶技巧

## 脚本编写最佳实践

### 实践 1：模块化设计

将复杂脚本拆分成多个函数，每个函数负责一个功能：

```python
def load_data(file_path):
    pass

def clean_data(data):
    pass

def analyze_data(data):
    pass

def save_result(result, output_path):
    pass

def main():
    data = load_data('input.xlsx')
    cleaned_data = clean_data(data)
    result = analyze_data(cleaned_data)
    save_result(result, 'output.xlsx')

if __name__ == '__main__':
    main()
```

### 实践 2：添加日志记录

使用 logging 模块记录脚本运行过程：

```python
import logging

logging.basicConfig(
    level=logging.INFO,
    format='%(asctime)s - %(levelname)s - %(message)s',
    handlers=[
        logging.FileHandler('script.log'),
        logging.StreamHandler()
    ]
)

logger = logging.getLogger(__name__)

def main():
    logger.info("开始处理数据")
    try:
        logger.info("数据处理完成")
    except Exception as e:
        logger.error(f"处理失败: {e}")

if __name__ == '__main__':
    main()
```

### 实践 4：错误处理

添加完善的错误处理机制：

```python
def process_file(file_path):
    try:
        with open(file_path, 'r') as f:
            data = f.read()
        return data
    except FileNotFoundError:
        print(f"❌ 文件不存在: {file_path}")
        return None
    except PermissionError:
        print(f"❌ 没有权限访问文件: {file_path}")
        return None
    except Exception as e:
        print(f"❌ 处理文件时出错: {file_path}, 错误: {e}")
        return None
```

### 技巧 1：分批处理大数据

```python
def process_large_file(file_path, batch_size=1000):
    with open(file_path, 'r') as f:
        batch = []
        for line in f:
            batch.append(line)
            
            if len(batch) >= batch_size:
                process_batch(batch)
                batch = []
        
        if batch:
            process_batch(batch)
```

from concurrent.futures import ThreadPoolExecutor

def process_file(file_path):
    pass

def parallel_process(files):
    with ThreadPoolExecutor(max_workers=4) as executor:
        results = executor.map(process_file, files)

### 技巧 4：缓存中间结果

```python
import json
import os

def get_cached_data(cache_file):
    if os.path.exists(cache_file):
        with open(cache_file, 'r') as f:
            return json.load(f)
    return None

def cache_data(data, cache_file):
    with open(cache_file, 'w') as f:
        json.dump(data, f)
```

### 注意事项 3：数据加密

```python
from cryptography.fernet import Fernet

def encrypt_data(data, key):
    f = Fernet(key)
    return f.encrypt(data.encode())

def decrypt_data(encrypted_data, key):
    f = Fernet(key)
    return f.decrypt(encrypted_data).decode()
```

A：按照以下步骤排查：
1. 查看错误信息，定位问题
2. 检查依赖库是否安装
3. 检查文件路径是否正确
4. 检查数据格式是否符合预期
5. 添加更多错误处理代码

A：尝试以下优化：
1. 使用更高效的算法
2. 分批处理大数据
3. 使用并行处理
4. 缓存中间结果
5. 使用更高效的库（如 numpy 替代纯 Python）
