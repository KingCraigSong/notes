# 文档整理

## 场景介绍

电脑上的文件越积越多，查找起来非常困难。本教程将介绍如何使用智能助手编写脚本，自动完成文件整理和归档工作。

## 常见文档整理任务

### 任务 1：文件分类整理

将文件夹中的文件按照类型分类到不同的文件夹中。

**提示词：**
```
请帮我写一个 Python 脚本，将文件夹中的文件按照类型分类。

【输入】
- 源文件夹路径：需要整理的文件夹

【分类规则】
- 文档类（.docx, .doc, .pdf, .txt, .md）→ 文档文件夹
- 表格类（.xlsx, .xls, .csv）→ 表格文件夹
- 图片类（.jpg, .png, .gif, .svg）→ 图片文件夹
- 演示类（.pptx, .ppt）→ 演示文件夹
- 代码类（.py, .js, .java, .html）→ 代码文件夹
- 其他类 → 其他文件夹

【输出】
- 在源文件夹中创建上述分类文件夹
- 将文件移动到对应的文件夹中
- 显示整理进度和统计信息

【要求】
- 如果目标文件夹不存在，自动创建
- 不要移动文件夹本身
- 保留原文件名
- 显示整理完成后的统计信息
```

**生成的脚本：**

```python
import os
import shutil

def organize_files(source_folder):
    category_map = {
        '文档': ['.docx', '.doc', '.pdf', '.txt', '.md'],
        '表格': ['.xlsx', '.xls', '.csv'],
        '图片': ['.jpg', '.jpeg', '.png', '.gif', '.svg'],
        '演示': ['.pptx', '.ppt'],
        '代码': ['.py', '.js', '.java', '.html', '.css']
    }

    if not os.path.exists(source_folder):
        print(f"❌ 错误：源文件夹不存在: {source_folder}")
        return

    file_count = 0
    category_counts = {}

    for filename in os.listdir(source_folder):
        file_path = os.path.join(source_folder, filename)
        
        if os.path.isfile(file_path):
            file_ext = os.path.splitext(filename)[1].lower()
            target_category = '其他'
            
            for category, extensions in category_map.items():
                if file_ext in extensions:
                    target_category = category
                    break
            
            target_folder = os.path.join(source_folder, target_category)
            if not os.path.exists(target_folder):
                os.makedirs(target_folder)
            
            shutil.move(file_path, os.path.join(target_folder, filename))
            file_count += 1
            category_counts[target_category] = category_counts.get(target_category, 0) + 1
            print(f"✓ {filename} → {target_category}")

    print("\n🎉 整理完成！")
    print(f"📦 共整理 {file_count} 个文件")
    for category, count in category_counts.items():
        print(f"  - {category}: {count} 个文件")

if __name__ == '__main__':
    source_folder = input("请输入需要整理的文件夹路径：")
    organize_files(source_folder)
```

### 任务 2：批量重命名

按照指定格式批量重命名文件。

**提示词：**
```
请帮我写一个 Python 脚本，批量重命名文件夹中的文件。

【输入】
- 文件夹路径：需要重命名的文件夹

【重命名规则】
- 格式：日期_序号_原文件名.扩展名
- 日期：文件修改日期（YYYYMMDD）
- 序号：同一日期内的顺序号（001, 002, ...）
- 原文件名：保留原文件名的主要部分

【要求】
- 只处理文件，不处理文件夹
- 显示重命名前后的文件名对比
- 显示重命名进度
- 完成后显示统计信息
```

**生成的脚本：**

```python
import os
import glob
from datetime import datetime

def batch_rename(folder_path):
    if not os.path.exists(folder_path):
        print(f"❌ 错误：文件夹不存在: {folder_path}")
        return

    files = [f for f in glob.glob(os.path.join(folder_path, '*')) if os.path.isfile(f)]
    
    date_groups = {}
    for file_path in files:
        modify_time = os.path.getmtime(file_path)
        date_str = datetime.fromtimestamp(modify_time).strftime('%Y%m%d')
        
        if date_str not in date_groups:
            date_groups[date_str] = []
        date_groups[date_str].append(file_path)

    renamed_count = 0
    for date_str, file_list in date_groups.items():
        file_list.sort()
        for idx, file_path in enumerate(file_list, 1):
            file_dir = os.path.dirname(file_path)
            file_ext = os.path.splitext(file_path)[1]
            original_name = os.path.splitext(os.path.basename(file_path))[0]
            
            new_name = f"{date_str}_{str(idx).zfill(3)}_{original_name}{file_ext}"
            new_path = os.path.join(file_dir, new_name)
            
            os.rename(file_path, new_path)
            renamed_count += 1
            print(f"✓ {os.path.basename(file_path)} → {new_name}")

    print(f"\n🎉 重命名完成！")
    print(f"📦 共重命名 {renamed_count} 个文件")

if __name__ == '__main__':
    folder_path = input("请输入文件夹路径：")
    batch_rename(folder_path)
```

### 任务 3：过期文件清理

自动清理指定天数前的文件。

**提示词：**
```
请帮我写一个 Python 脚本，清理指定天数前的文件。

【输入】
- 文件夹路径：需要清理的文件夹
- 保留天数：保留多少天内的文件（默认 30 天）

【清理规则】
- 删除指定天数前修改的文件
- 不删除文件夹
- 不删除指定扩展名的文件（如 .pdf, .doc）

【要求】
- 显示即将删除的文件列表
- 询问用户是否确认删除
- 显示删除进度
- 完成后显示统计信息
```

**生成的脚本：**

```python
import os
import glob
from datetime import datetime, timedelta

def clean_expired_files(folder_path, days_to_keep=30):
    if not os.path.exists(folder_path):
        print(f"❌ 错误：文件夹不存在: {folder_path}")
        return

    cutoff_date = datetime.now() - timedelta(days=days_to_keep)
    protected_extensions = ['.pdf', '.doc', '.docx', '.xlsx', '.xls']
    
    files_to_delete = []
    for file_path in glob.glob(os.path.join(folder_path, '**/*'), recursive=True):
        if os.path.isfile(file_path):
            modify_time = datetime.fromtimestamp(os.path.getmtime(file_path))
            file_ext = os.path.splitext(file_path)[1].lower()
            
            if modify_time < cutoff_date and file_ext not in protected_extensions:
                files_to_delete.append(file_path)

    if not files_to_delete:
        print("✅ 没有需要删除的过期文件")
        return

    print(f"⚠️ 即将删除以下 {len(files_to_delete)} 个文件：")
    for file_path in files_to_delete[:5]:
        print(f"  - {file_path}")
    if len(files_to_delete) > 5:
        print(f"  ... 还有 {len(files_to_delete) - 5} 个文件")

    confirm = input("\n是否确认删除？(y/n): ").lower()
    if confirm != 'y':
        print("取消删除操作")
        return

    deleted_count = 0
    for file_path in files_to_delete:
        os.remove(file_path)
        deleted_count += 1
        print(f"✓ 删除: {file_path}")

    print(f"\n🎉 清理完成！")
    print(f"🗑️ 共删除 {deleted_count} 个文件")

if __name__ == '__main__':
    folder_path = input("请输入文件夹路径：")
    days = int(input("请输入保留天数（默认 30）：") or 30)
    clean_expired_files(folder_path, days)
```

### 任务 4：内容提取

从文件中提取关键信息并生成摘要。

**提示词：**
```
请帮我写一个 Python 脚本，从多个文件中提取内容并生成摘要。

【输入】
- 文件夹路径：包含多个文件
- 支持的文件类型：.txt, .md, .docx, .pdf

【输出】
- 摘要文件：包含每个文件的关键信息
- 内容包括：文件名、文件大小、修改时间、内容摘要

【要求】
- 自动识别文件类型并使用相应的提取方法
- 摘要长度限制在 200 字以内
- 显示提取进度
- 完成后显示统计信息
```

## 注意事项

1. **备份数据**：在运行整理脚本前，建议备份原始数据
2. **测试运行**：先在测试文件夹中测试脚本，确保效果符合预期
3. **谨慎删除**：清理过期文件时，确保没有误删重要文件
4. **保留原文件**：重命名操作前，确保有原文件的备份
