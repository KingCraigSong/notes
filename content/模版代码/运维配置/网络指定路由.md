# Linux 网络指定路由配置

## 添加静态路由

```bash
sudo route add -net 10.60.0.0 -netmask 255.255.0.0 10.60.226.254
```

## 参数说明

- `-net`：指定目标网络地址
- `-netmask`：指定子网掩码
- 最后一个参数：指定网关地址

## 查看路由表

```bash
route -n
```

## 删除路由

```bash
sudo route del -net 10.60.0.0 -netmask 255.255.0.0
```

## 永久生效

以上命令在系统重启后会失效，如需永久生效，需要添加到网络配置文件中：

**CentOS/RHEL**：
编辑 `/etc/sysconfig/network-scripts/route-eth0` 文件，添加：
```
10.60.0.0/16 via 10.60.226.254
```

**Ubuntu/Debian**：
编辑 `/etc/network/interfaces` 文件，在对应接口配置中添加：
```
up route add -net 10.60.0.0/16 gw 10.60.226.254
```
