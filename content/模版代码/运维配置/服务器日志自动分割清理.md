# 服务器日志自动分割与清理

## 分割清理脚本

创建 `auto_clean.sh` 文件：

```bash
#!/bin/bash 

DIR=/opt/soft/log
d=$(date +%Y%m%d) 
cd $DIR 
cp catalina.out catalina.out.bak${d} 
echo "" > catalina.out 
find ./ -mtime +7 -name "catalina.out.bak*" -exec rm -rf {} \;
```

### 脚本说明

| 命令 | 作用 |
| :--- | :--- |
| `DIR=/opt/soft/log` | 设置日志目录路径 |
| `date +%Y%m%d` | 获取当前日期，格式为年月日 |
| `cp catalina.out catalina.out.bak${d}` | 备份当前日志文件 |
| `echo "" > catalina.out` | 清空原日志文件 |
| `find ./ -mtime +7 -name "catalina.out.bak*" -exec rm -rf {} \;` | 删除7天前的备份文件 |

### 脚本优化版

以下是更完善的版本，增加了日志记录和错误处理：

```bash
#!/bin/bash

LOG_DIR="/opt/soft/log"
BACKUP_PREFIX="catalina.out.bak"
DAYS_TO_KEEP=7
SCRIPT_LOG="/var/log/log_cleanup.log"

# 检查日志目录是否存在
if [ ! -d "$LOG_DIR" ]; then
    echo "$(date '+%Y-%m-%d %H:%M:%S') - 错误: 日志目录 $LOG_DIR 不存在" >> "$SCRIPT_LOG"
    exit 1
fi

cd "$LOG_DIR" || exit 1

# 获取当前日期
CURRENT_DATE=$(date +%Y%m%d)

# 备份日志文件
if [ -f "catalina.out" ]; then
    cp catalina.out "${BACKUP_PREFIX}${CURRENT_DATE}"
    if [ $? -eq 0 ]; then
        echo "$(date '+%Y-%m-%d %H:%M:%S') - 成功备份日志到 ${BACKUP_PREFIX}${CURRENT_DATE}" >> "$SCRIPT_LOG"
    else
        echo "$(date '+%Y-%m-%d %H:%M:%S') - 错误: 备份日志失败" >> "$SCRIPT_LOG"
        exit 1
    fi

    # 清空原日志文件
    echo "" > catalina.out
    if [ $? -eq 0 ]; then
        echo "$(date '+%Y-%m-%d %H:%M:%S') - 成功清空 catalina.out" >> "$SCRIPT_LOG"
    else
        echo "$(date '+%Y-%m-%d %H:%M:%S') - 错误: 清空 catalina.out 失败" >> "$SCRIPT_LOG"
        exit 1
    fi
else
    echo "$(date '+%Y-%m-%d %H:%M:%S') - 警告: catalina.out 文件不存在" >> "$SCRIPT_LOG"
fi

# 删除过期备份文件
find ./ -mtime +"$DAYS_TO_KEEP" -name "${BACKUP_PREFIX}*" -exec rm -rf {} \;
if [ $? -eq 0 ]; then
    echo "$(date '+%Y-%m-%d %H:%M:%S') - 成功删除 ${DAYS_TO_KEEP} 天前的备份文件" >> "$SCRIPT_LOG"
else
    echo "$(date '+%Y-%m-%d %H:%M:%S') - 错误: 删除过期备份文件失败" >> "$SCRIPT_LOG"
    exit 1
fi

exit 0
```

## 定时任务配置

### 编辑 crontab

```bash
crontab -e
```

### 添加任务

设置每天23点59分执行 `auto_clean.sh` 文件进行数据清理任务：

```bash
59 23 * * * /opt/soft/log/auto_clean.sh >/dev/null 2>&1
```

### 定时任务格式说明

| 字段 | 含义 | 取值范围 |
| :--- | :--- | :--- |
| 第1个字段 | 分钟 | 0-59 |
| 第2个字段 | 小时 | 0-23 |
| 第3个字段 | 日期 | 1-31 |
| 第4个字段 | 月份 | 1-12 |
| 第5个字段 | 星期 | 0-7（0和7都表示周日） |

### 常用定时任务示例

```bash
# 每天凌晨3点执行
0 3 * * * /path/to/script.sh

# 每周一凌晨3点执行
0 3 * * 1 /path/to/script.sh

# 每月1号凌晨3点执行
0 3 1 * * /path/to/script.sh

# 每小时执行一次
0 * * * * /path/to/script.sh

# 每10分钟执行一次
*/10 * * * * /path/to/script.sh
```

### 查看定时任务

```bash
crontab -l
```

### 删除定时任务

```bash
crontab -r
```
